# Corrector-de-texto
**Alumno: Angelo Alvarez**

Este programa es un corrector de texto rudimentario.

## Build
Para compilar el programa:

```
$ make
```

# Funcionamiento
Para corregir el texto de un archivo de texto
```
$ ./main arch_text salida cache
```

El archivo se corregira en salida. Cache es un archivo de cache,
donde se guardaran las palabras ya corregidas.
